UC13. Marxar del grup

Criteris d'acceptació 0, 1, 2, 3

0: Si no es troba cap grup amb el nom especificat el sistema mostrara el missatge: "No s'ha trobat cap grup amb aquest nom" 1: Si l'usuari s'elimina correctament del grup el sistema mostrara el missatge: "S'ha eliminat al membre satisfactoriament"

Example del test d'acceptació 13.1
La següent linia mostra els resultats que esperem si demanem al sistema eliminar un usuari d' un grup amb nom Grup5 i amb el nom d'usuari ajaleo@gmail.com si hi ha el Grup5 creat pero el nom del grup no coincideix amb cap grup creat L'usuari no pertany al grup
