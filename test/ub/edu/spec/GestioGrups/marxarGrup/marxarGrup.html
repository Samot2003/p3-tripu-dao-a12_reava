UC13. Marxar del grup

Criteris d'acceptació 0, 1, 2, 3

0: Si no es troba cap grup amb el nom especificat el sistema mostrara el missatge: "No s'ha trobat cap grup amb aquest nom" 1: Si no es troba usuari amb el nom especificat el sistema mostrara el missatge: "L' usuari no ha sigut trobat a la base de dades" 2: Si es troba el grup desitjat pero l'usuari indicat no hi pertany: "S'ha trobat el grup pero l'usuari no pertany al grup" 3: Si l'usuari s'elimina correctament del grup el sistema mostrara el missatge: "S'ha eliminat al membre satisfactoriament"

Example del test d'acceptació 13.1
La següent linia mostra els resultats que esperem si demanem al sistema eliminar un usuari d' un grup amb nom Grup5 i amb el nom d'usuari ajaleo@gmail.com si hi ha el Grup5 creat pero el nom del grup no coincideix amb cap grup creat No s'ha trobat cap grup amb aquest nom
Example del test d'acceptació 13.2
La següent linia mostra els resultats que esperem si demanem al sistema eliminar un usuari d' un grup amb nom Grup3 i amb el nom d'usuari peepo@gmail.com si hi ha el Grup3 creat pero el nom del usuari no coincideix amb cap usuari registrat L' usuari no ha sigut trobat a la base de dades.
Example del test d'acceptació 13.3
La següent linia mostra els resultats que esperem si demanem al sistema eliminar un usuari d' un grup amb nom Grup3 i amb el nom d'usuari dtomacal@yahoo.cat si hi ha el Grup3 creat pero aquest usuari no pertany al grup S'ha trobat el grup pero l'usuari no pertany al grup.
Example del test d'acceptació 13.4
La següent linia mostra els resultats que esperem si demanem al sistema eliminar un usuari d' un grup amb nom Grup3 i amb el nom d'usuari dtomacal@yahoo.cat si hi ha el Grup3 creat S'ha eliminat al membre satisfactoriament.
