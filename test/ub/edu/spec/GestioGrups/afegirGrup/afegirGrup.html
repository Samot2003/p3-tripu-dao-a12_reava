UC12. Afegir-se al grup

Criteris d'acceptació 0, 1, 2, 3

0: Si no hi han grups el sistema mostrara el missatge: "No hi ha cap grup a la llista afegeix algun primer" 1: Si no es troba cap grup amb el nom especificat el sistema mostrara el missatge: "No s'ha trobat cap grup amb aquest nom" 2: Si no es troba usuari amb el nom especificat el sistema mostrara el missatge: "L' usuari no ha sigut trobat a la base de dades" 3: Si l'usuari s'afegeix correctament al grup el sistema mostrara el missatge: "S'ha agregat el membre satisfactoriament"
