US15.1: Valorar Punts De Control Dislikes

Criteris d'acceptació 0, 1

0: Si el ID del punt de pas entrat no coincideix amb cap ID, es mostrarà el missatge: "No s'ha trobat el punt de control" 1: Si el punt de pas es valora correctament, es mostrarà el missatge: "Punts de control valorats correctament"

Example del test d'acceptació 15.0
La següent linia mostra els resultats que esperem si demanem al sistema valorar un punt de pas on el ID d 'aquest no coincideix amb cap dels que es tenen.p2 1. No s'ha trobat el punt de control
Example del test d'acceptació 15.1
La següent linia mostra els resultats que esperem si demanem al sistema valorar un punt de pas on el ID d 'aquest no coincideix amb cap dels que es tenen.p1 1. Punts de control valorats correctament
