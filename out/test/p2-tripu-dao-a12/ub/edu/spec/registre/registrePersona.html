Com a persona no enregistrada

Vull registrar-me a l'aplicació amb les dades corresponents

Per a ser persona de TripUB i accedir als continguts de l’aplicació

Acceptance Criteria 1.1: Registre Persona: Identificador de persona (correu) ja existent

El sistema dona opció d'enregistrar però el seu correu ja existeix en la llista de socis

Example del test d'acceptació 1.1
Posem una persona ja existent: ajaleo@gmail.com el sistema hauria de donar Persona ja existent en el Sistema.

Acceptance Criteria 1.2: Registre Persona: Contrassenya no segura

El sistema dona opció d'enregistrar però la contrassenya no és prou segura

Example del test d'acceptació 1.2

Posem un usuari qualsevol no existent ajaleonou@gmail.com i una contrassenya no segura: werty, el sistema hauria de donar Contrassenya no prou segura.

Posem un usuari qualsevol no existent ajaleonou@gmail.com i una contrassenya segura: Qwertyuiop9, el sistema hauria de donar Contrassenya segura.

Acceptance Criteria 1.3: Registre Persona: Correu electronic no vàlid

El sistema dóna l'opció d'enregistrar però el nom no té el format d'un correu electrònic

Posem un usuari qualsevol amb un format incorrecte en el mail ajaleonougmail.com , el sistema hauria de donar Correu en format incorrecte.

Posem un usuari qualsevol amb un format incorrecte en el mail ajaleonou@gmailcom, el sistema hauria de donar Correu en format incorrecte.

Posem un usuari qualsevol amb format correcte ajaleonou@gmail.com, el sistema hauria de donar Correu en format correcte.

Acceptance Criteria 1.4: Registre Persona: Registres correctes

El sistema dona opció d'enregistrar i els registres són correctes
